# Changelog

Todos los cambios notables de este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2025-11-30

### Añadido
- Sistema completo de gestión de tareas con metodología Scrum
- Tablero Kanban interactivo con drag & drop
- Gestión de sprints con métricas y estadísticas
- Dashboard con visualización de datos en tiempo real
- Sistema de autenticación con JWT
- API REST completa con Node.js + Express
- Frontend moderno con React 18 + Vite + Tailwind CSS
- Base de datos JSON (preparada para migración futura)
- Suite completa de pruebas unitarias con Jest
- Pruebas E2E automatizadas con Selenium WebDriver
- Pipeline CI/CD con GitHub Actions
- Documentación completa de GitHub Flow
- Templates para Issues y Pull Requests
- Sistema de gestión de usuarios y roles
- Filtrado avanzado de tareas
- Responsive design para móviles y tablets

### Características del Diseño
- Interfaz moderna y minimalista
- Animaciones suaves y transiciones
- Sistema de colores coherente
- Iconos con Lucide React
- Componentes reutilizables
- Dark mode ready (preparado para tema oscuro)

### Testing
- Pruebas unitarias backend (Jest + Supertest)
- Pruebas unitarias frontend (Vitest + React Testing Library)
- Pruebas E2E (Selenium WebDriver)
- Cobertura de código >80%

### Documentación
- README completo con instrucciones de instalación
- Guía detallada de GitHub Flow
- Guía de contribución
- Templates de Issues
- Template de Pull Requests
- Comentarios en código

### Configuración
- GitHub Actions para CI/CD
- ESLint y Prettier ready
- Git hooks con Husky (preparado)
- Entorno de desarrollo optimizado

### ️ Tecnologías
- **Frontend**: React 18, Vite, Tailwind CSS, Zustand, React Router, Axios
- **Backend**: Node.js, Express, JWT, bcryptjs
- **Testing**: Jest, Vitest, Selenium, React Testing Library
- **CI/CD**: GitHub Actions
- **Otros**: date-fns, lucide-react, react-hot-toast
